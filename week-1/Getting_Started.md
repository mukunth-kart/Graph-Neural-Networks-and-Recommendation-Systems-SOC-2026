## Steps to Take

1. **Fork and clone** the repository to your local machine, checkout a new branch, and get started.
2. **Go through the notebooks.** If you hit a roadblock anywhere, check the official docs or reach out to the mentors.
3. **Alternative approach:** Watch the core videos first, then work through the corresponding notebooks side-by-side.
4. **Ready for a challenge?** If the notebooks are already completely familiar to you, jump straight to the assignments [here](assignment/Assignment.md).

## Environment Setup

Your setup workflow depends on whether you have access to a local GPU:

* **With a Local GPU:** Set up [Miniconda](https://www.youtube.com/watch?v=HwkHIan39W8) as your environment manager if you haven't already. Use **Python 3.11** if you are unsure which version to pick—it is currently the most stable for data science packages. *Note: While we use notebooks for learning, we highly encourage writing your assignments in pure `.py` files to build good scripting habits.*
* **Without a Local GPU:** Use [Google Colab](https://www.youtube.com/watch?v=FXKMmilL70w) to utilize its free cloud GPU. While a GPU isn't strictly necessary for Week 1 tasks, you will definitely need one as we move ahead. Running `.py` files in Colab can be a bit tedious, but give it a shot if you can!
* **New to Git?** If concepts like "forking" or "cloning" sound totally foreign, check out [How Git Works](https://www.youtube.com/watch?v=e9lnsKot_SQ) and [Git Basics](https://www.youtube.com/watch?v=mJ-qvsxPHpY) to get up to speed.

> 💡 **Tip:** LLMs (like Gemini or ChatGPT) are incredibly efficient at walking you through local environment setup and Git troubleshooting. If you get stuck on an installation error, try asking them first! For lingering issues, contact your [Mentor via WhatsApp](https://web.whatsapp.com/).

## A Bit of Unsolicited Advice...

Don't spend hours debugging cosmetic details in Matplotlib or Seaborn. It’s good to understand how they work, but focus heavily on knowing **which** plot to use for your data and **why**. In modern data science, conceptual understanding, math, and data intuition will always trump memorizing code syntax.
